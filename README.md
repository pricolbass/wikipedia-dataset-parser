# wikipedia dataset parser

Потоковый парсер XML-дампов Википедии для извлечения графа гиперссылок и Ground Truth (категорий). Оптимизирован для работы с графами масштаба 1 млн+ узлов на стандартном железе (16 GB RAM) без загрузки дампа в память.

Разработан для исследования алгоритмов обнаружения сообществ (Leiden) и эмпирической проверки теоремы Фортунато-Бартелеми на графах знаний.

## Возможности
* **Потоковая обработка:** Читает `.xml.bz2` напрямую, RAM не зависит от размера дампа.
* **Многоуровневая фильтрация:** Блэклисты, отсев заготовок (stubs), тематический отбор.
* **Гибкий Ground Truth:** Генерация эксклюзивной (стратегия «первая тема» для NMI/ARI) и перекрывающейся разметки.
* **Готовый pipeline:** От парсинга до топологического анализа и многоуровневой оценки кластеризации.

## Установка

```bash
git clone https://github.com/pricolbass/wikipedia-dataset-parser.git
cd wikipedia-dataset-parser
pip install -r requirements.txt
```

**Зависимости:** `igraph`, `pandas`, `numpy`, `scikit-learn`, `scipy`, `matplotlib`, `tqdm`.

## Использование

1. Скачайте официальный дамп (например, `ruwiki-latest-pages-articles.xml.bz2`) и укажите путь к нему в `parser.py`.
2. **Парсинг и сборка графа:**
   ```bash
   python parser.py
   ```
3. **Топологический анализ** (степени, кластеризация, степенной закон):
   ```bash
   python main_analysis.py
   ```
4. **Кластеризация (Leiden) и оценка качества:**
   ```bash
   python main_leiden.py
   ```
   *Поддерживает аргументы, например: `python main_leiden.py --resolutions 1.0 2.0`*

## Структура проекта

| Файл | Назначение |
| :--- | :--- |
| `parser.py` | Потоковый парсер XML-дампа, фильтрация, сохранение `.edgelist` и `.cmty`. |
| `config.py` | Централизованная конфигурация (пути, `k_min`, диапазоны размеров, параметры Leiden). |
| `graph_loader.py` | Загрузка графа и GT в `igraph`, валидация, стратегия «первая тема». |
| `analysis.py` | Расчет топологии, MLE-оценка показателя $\gamma$ (степенной закон). |
| `leiden_runner.py` | Запуск Leiden, расчет multi-level NMI/ARI, проверка статистических гипотез. |
| `main_analysis.py` | Точка входа для топологического анализа (генерирует `report.json`). |
| `main_leiden.py` | Точка входа для экспериментов с кластеризацией (генерирует `multi_resolution_*.json`). |
